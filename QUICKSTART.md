# 🚀 快速启动指南

## 第一步：安装依赖

双击运行 `install.bat`，系统会自动：
1. 检查 Python 环境
2. 安装所有依赖包
3. 验证配置文件

**预计时间**: 2-5分钟

## 第二步：启动平台

双击运行 `run.bat`

**预计时间**: 10-30秒

## 第三步：开始分析

1. 在左侧边栏输入股票代码（如 `000001.SZ`）
2. 选择分析周期（默认90天）
3. 勾选需要的分析选项
4. 点击页面任意位置

## 常见问题速查

### ❌ 启动失败

```bash
# 检查Python版本
python --version  # 需要 3.9-3.12

# 重新安装依赖
pip install -r requirements.txt --upgrade

# 手动启动
streamlit run app.py
```

### ❌ API调用失败

1. 检查 `.env` 文件是否存在
2. 检查 API Key 是否正确
3. 检查网络连接

### ❌ 数据获取失败

1. 检查股票代码格式：
   - 深圳：`000001.SZ`
   - 上海：`600519.SH`
2. 检查股票是否上市
3. 检查 Tushare Token 是否有效

## 📊 界面说明

### 主界面
- **顶部**: 平台标题
- **左侧**: 分析设置
- **右侧**: 分析结果

### 分析结果
1. **关键指标卡片**: 价格、涨跌幅、MA20、RSI
2. **价格走势图**: 收盘价、均线、布林带
3. **MACD图表**: MACD和信号线
4. **RSI图表**: 相对强弱指标
5. **成交量图**: 成交量及均线
6. **详细数据表**: 所有技术指标
7. **AI分析**: 智能投资建议
8. **基本面数据**: 财务报表
9. **选股建议**: AI推荐股票

## 💡 使用技巧

### 1. 多股票对比
- 输入多个股票代码（用空格分隔）
- 一次分析多只股票

### 2. 不同周期分析
- 短线：30-60天
- 中线：90-180天
- 长线：180-365天

### 3. 技术指标组合
- 趋势跟踪：MA + MACD
- 波动分析：布林带 + RSI
- 量价分析：成交量 + MA

### 4. AI分析优化
- 简单：快速概览
- 中等：详细分析
- 详细：深度解读

## 🔧 高级配置

### 使用虚拟环境

```bash
# 1. 创建虚拟环境
python -m venv venv

# 2. 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动平台
streamlit run app.py
```

### 自定义主题

编辑 `app.py` 中的CSS部分，可以自定义颜色和样式。

### 修改AI分析内容

编辑 `app.py` 中的 `analyze_with_zhipu()` 函数。
