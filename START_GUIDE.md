## 🔧 常见问题

### Q: 启动失败怎么办？

### Q: AI分析失败？
1. 检查网络连接
2. 检查智谱AI API Key是否有效
3. 查看错误提示

### Q: 数据获取失败？
1. 检查股票代码格式（`.SZ` 或 `.SH`）
2. 检查股票是否上市
3. 检查 Tushare Token 是否有效

## 💡 使用技巧

- 波动分析：布林带 + RSI
- 量价分析：成交量 + MA
